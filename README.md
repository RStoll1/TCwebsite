# TC Website

Simple Next.js storefront prototype using BigCartel product data.

This is an early-stage band website project intended to become a merch funnel.

## Built so far

- Fetches products from BigCartel
- Normalizes API data into a typed `Product` model
- Renders products as responsive cards on the marketing page

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- BigCartel Products API

## Note

If the BigCartel store is in maintenance mode, product fetches will fail.
